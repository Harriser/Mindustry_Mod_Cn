### 说明
- 由于6.0版本的较大更新，导致104版本mod部分或大部分失效，基于对mod的热爱而编辑本页面。
- 由于采用构建或沙盒模式测试内容，因此不保证在（包括但不仅限于）战役模式、PVP或生存地图稳定性。
- 翻译内容来自有道词典和谷歌，加上自测而来，部分内容可能表述不完整或不准确
- 6.0后修改了部分建筑或资源名称，描述中优先使用104版本进行说明
- 适用版本：6.0 Build 120
- 测试环境版本：6.0 Build 121.2
